# Операторы сравнения

| Оператор сравнения | Описание         |
| ------------------ | ---------------- |
| ==                 | Равно            |
| !=                 | Не равно         |
| <                  | Меньше           |
| >                  | Больше           |
| <=                 | Меньше или равно |
| >=                 | Больше или равно |

## Сравнение строк <a href="stringcomparison" id="stringcomparison"></a>

Строки сравниваются в лексикографическом порядке

"abc" == "abc"

"abc" < "ac"

"abc" > "ab"
