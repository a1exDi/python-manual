# Операторы сравнения

| Оператор сравнения | Описание |
| :--- | :--- |
| == | Равно |
| != | Не равно |
| &lt; | Меньше |
| &gt; | Больше |
| &lt;= | Меньше или равно |
| &gt;= | Больше или равно |

## Сравнение строк <a id="stringcomparison"></a>

Строки сравниваются в лексикографическом порядке

"abc" == "abc"

"abc" &lt; "ac"

"abc" &gt; "ab"
