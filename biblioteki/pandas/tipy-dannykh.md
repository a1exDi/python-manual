# Типы данных



| \_                    | Тип в Python | Тип в Pandas |
| --------------------- | ------------ | ------------ |
| Строка                | str          | object       |
| Целые числа           | int          | int64        |
| Вещественные числа    | float        | float64      |
| Логический тип данных | bool         | bool         |
